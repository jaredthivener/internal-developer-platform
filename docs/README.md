# Documentation Site

This site is built with the latest Docusaurus classic TypeScript template and publishes to GitHub Pages.

## Installation

```bash
npm install
```

## Local Development

```bash
npm start
```

This starts the documentation site locally with live reload.

## Build

```bash
npm run build
```

This generates the static site into the `build` directory.

## Deployment

Deployment is handled by the repository GitHub Actions workflow for GitHub Pages.

The site is configured for:

- URL: `https://jaredthivener.github.io`
- Base path: `/internal-developer-platform/`
